# Задание 5: Анкета разработчика с авторизацией

## Описание
Веб-форма для регистрации разработчиков с сохранением данных в MySQL, авторизацией и возможностью редактирования.

## Технологии
- PHP 7+
- MySQL/MariaDB
- HTML5/CSS3
- Сессии PHP для авторизации
- Cookies для сохранения данных

## Структура БД
- `users` - пользователи (логин, хеш пароля)
- `applications` - заявки
- `programming_languages` - справочник языков
- `application_languages` - связь заявок и языков

## Функционал
1. Регистрация с автоматической генерацией логина и пароля
2. Вход по логину и паролю
3. Редактирование данных после авторизации
4. Валидация всех полей регулярными выражениями
5. Сохранение данных в Cookies

## Установка
1. Импортировать db.sql в MySQL
2. Настроить подключение к БД в index.php и login.php
3. Открыть index.php в браузере
